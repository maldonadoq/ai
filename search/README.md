# Search in graphs

Visualization of paths from different search methods.

## Required Dependencies
 * OpenGL
 * Glut
 * C++

## Compile
 * $ cd folder
 * $ g++ -std=c++11 graph.cpp -o graph.out -lGL -lGLU -lglut
